# FloatRec

macOS 메뉴바 기반 화면 녹화 도구의 초기 프로토타입입니다.

현재 구현 범위:

- 메뉴바에서 녹화 시작/종료 상태 전환
- 글로벌 단축키 `⌘⇧9`
- 화면 녹화 권한 요청
- ScreenCaptureKit 기반 디스플레이/윈도우 소스 목록 로드
- macOS 15 이상에서 디스플레이/윈도우 실녹화 경로 연결
- 전체 화면 오버레이 기반 영역 선택
- 디스플레이/영역 녹화의 커서 추적 기반 자동 줌 후처리
- 클릭 리플 강조 후처리
- Shelf 카드 썸네일과 별도 미리보기 창
- 녹화 종료 후 우하단 `Floating Shelf`
- Shelf 카드에서 저장, 공유, Finder 열기, 드래그

주의:

- 영역 모드는 선택한 화면의 `sourceRect`를 사용해 녹화합니다.
- 자동 줌은 현재 디스플레이/영역 모드에 우선 적용되고, 윈도우 모드는 원본을 유지합니다.
- 클릭 강조는 현재 후처리 가능한 디스플레이/영역 모드에서 가장 자연스럽게 적용됩니다.
- 실녹화 실패 시에는 종료 흐름 검증용 데모 클립으로 fallback 합니다.

실행:

```bash
swift run
```

DMG 빌드:

```bash
./scripts/build_dmg.sh
```

산출물:

- `dist/FloatRec.dmg`
- `dist` 산출물은 git 추적에서 제외됩니다.
